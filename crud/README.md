# CRUD APPLICATION WITH NODE AND REACT

this is a simple application to manage employees

## functionalities
* create employee
* get all employees
* get single employee
* update employee 
* delete employee

![screenshot](images/final.png)

# front end
- HTML5
- CSS3
- bootstrap
- reactjs
# backend 
- nodejs
- express
- sequlize
- sequelize-cli
- mysql 

# data management 
- context api with Hooks
